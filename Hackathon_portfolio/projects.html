Data scraping and Incremental loading in VBA

I needed to create a way to get automatic forex rates for a client and have a table that would automatically add multiple Exchange rates daily which would then be sourced into a dashboard
I could not use Power Query and the Append does not have an incremental load function where the table can increase and adjust overtime with out intervention. Using VBA, and CLI I created a batch file and instructed the Excel file to execute the batch file on my computer
This means the VBA has access to task scheduler and can open the Excel, load the rates and close it a set times.
I was very proud of this project as it required me to think outside of the box
